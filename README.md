У даній лабораторній роботі реалізовано обробку XML-документу з використанням SAX та DOM парсерів.
В якості вхідних даних використовується датасет популярних імен дітей у місті Нью-Йорк.

XML-файл містить інформацію про:
- рік народження;
- стать дитини;
- етнічну приналежність;
- ім’я;
- кількість дітей з цим ім’ям;
- рейтинг популярності.



Реалізований функціонал

У ході виконання роботи було реалізовано:

- аналіз структури XML-документу за допомогою SAX-парсера;
- отримання списку всіх тегів;
- створення XSD-схеми та валідація XML;
- отримання списку етнічних груп;
- вибір найпопулярніших імен для заданої етнічної групи;
- сортування імен за рейтингом;
- створення нового XML-документу (DOM);
- читання створеного XML-документу.
